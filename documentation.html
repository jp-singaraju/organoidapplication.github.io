Instructions

1. Select all the specific functions that need to be executed.
2. The buttons and features of the selected function boxes will become available.
3. Go through each function box one by one (more information below).
4. Follow the buttons and watch the progress bar at the bottom to determine the progress.
5. Before starting a new function, wait for the progress bar to finish and a updated folder to pop-up.
6. To run more operations, close the application and reopen.
