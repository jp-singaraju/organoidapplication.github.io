Organoid Application Documentation

Version 1.2

Instructions

1. Select all the specific functions that need to be executed.
2. The buttons and features of the selected function boxes will become available.
3. Go through each function box one by one (more information below).
4. Follow the buttons and watch the progress bar at the bottom to determine the progress.
5. Before starting a new function, wait for the progress bar to finish and a updated folder to pop-up.
6. To run more operations, close the application and reopen.

Brightness/Contrast Function

1. Open the folder with either .jpg/.tif images and click to view images.
2. The images will load into the box on in the middle.
3. Update the brightness/contrast (if needed), then hit the update button.
4. After all images have been updated to user preferences, hit process and wait for the function to finish.
5. Once the function is finished, a new folder will pop-up with the new images (.../...Filtered/).

Remove Noise Function

1. Open the folder with either .jpg/.tif images.
2. Hit the process button to perform the noise removal and view the progress bar for updates.
3. Once the function is finished, a new folder will pop-up with the new images (.../...Noise Removed/).

Fractal Analysis Function

1. Open the folder with either .jpg/.tif images.
2. Hit the process button to perform the fractal analysis calculations and view the progress bar for updates.
3. Once the function is finished, a new folder will pop-up with the new images (.../...Fractal Images).
4. Within this new folder, there will be the fractal images, as well as a .xlsx file with the fractal values for each image.

Area/Intensity Function

1. Open the folder with either .jpg/.tif images.
2. Hit the process button to perform the area/intensity calculations and view the progress bar for updates.
3. Once the function is finished, the original folder will pop-up with the original images.
4. Within this original folder, there will be the original images, as well as a .xlsx file with the area/intensity calculations.
5. The calculations are explained below.
Organoid Area - The pixel area that the organoid covers in the image. Total Image Area - The pixel area of the total image. Percentage of Image Covered by Organoid - The pixel area of the organoid divided by the pixel area of the total image. Average Organoid Intensity per Pixel - The total intensity of the organoid divided by the pixel area of the organoid. Organoid Area * Average Organoid Intensity per Pixel - The total intensity of the organoid.
